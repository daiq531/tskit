# tskit

This project is for creating one small docker image which include troubleshooting tools, performance tools and even small pieces of python code to do some basic validation which can't be met by other tools.

Alpine linux image is used for building this trouble shooting kit. Below tools have already been included in it:
-- ping 
-- ip

Alpine linux use apk tools to install packages. Below show its simple usage:
apk install python

## tool list to be installed
-- python
-- iperf
